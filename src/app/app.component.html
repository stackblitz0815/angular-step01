Hello Angular

This is a minimal Angular application!

What you see here is the output of app.component.html

The application is structured in overall styles, modules, components and services.
This minimal application has no service.

The general styles, that are applied to the whole app are in the file styles.scss

The entry point to the application is the index.html file in which the component app is displayed (<app></app>).
The main.ts file remains untouched it defines the bootstrapping (loading the app.module.ts).

A module app.module.ts bundles components that belopng together.
This application has exactly one component named app.

A component consists of 3 files:

  • component-name.component.html - the components template - the UI.
  • component-name.component.scss - the components styles (sass).
    These styles are only applied to the component
  • component-name.component.ts - the components typescript code - the control logic.
